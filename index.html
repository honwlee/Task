Task

this is an example of qnet's utilhub app.

generate tool: generator-qapp

backend server: http://backend.ihudao.dt.hudaokeji.com

frontend sdk: http://ude.utilhub.dt.hudaokeji.com

components server: http://center.utilhub.dt.hudaokeji.com

Init

$ cd ./build; npm install; cd ..;
$ cd ./test; npm install; cd ..;
$ cd ./src/lib; bower install; cd ../..;

Start Test Server

$ cd ./test; node server.js

Access Test Server

put this link http://localhost:9000 to your favourite browser.
